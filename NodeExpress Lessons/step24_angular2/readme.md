Instructions

Install Node:
https://nodejs.org/en/

Install TypeScript:
http://www.typescriptlang.org/

npm install typescript -g

Install VS Code: 
https://code.visualstudio.com/

Install TSD:
https://github.com/DefinitelyTyped/tsd

npm install tsd -g

Install Node packages:
npm install

Run Server:
npm start

View in Browser:
http://localhost:3000
